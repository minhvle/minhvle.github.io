Synopsis:
======
This unit will introduce advanced concepts in the areas of database design, including document-store, column-store, and graph database design; implementations in non-relational database systems.


Outcomes
======
On successful completion of this unit, you should be able to:

1. Describe various non-relational database systems, including NoSQL;
2. Compare and contrast between relational and non-relational database design and model;
3. Design database systems using document-store and column-store design techniques;
4. Explain transactions systems in non-relational systems;
5. Implement and manipulate document-store and column-store database systems;
6. Investigate graph model in database systems;
7. Demonstrate graph query processing.
